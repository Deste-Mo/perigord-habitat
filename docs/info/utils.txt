CAHIER DES CHARGES COMPLET Plateforme nationale QUI FAIT QUOI 

1. Objectif du projet 
Créer une plateforme numérique appelée :
QUI FAIT QUOI
destinée à tous les bailleurs de France.

L’outil permet aux locataires de comprendre rapidement :
qui doit intervenir qui paie pourquoi comment diagnostiquer comment dépanner 
lorsqu’un problème apparaît dans leur logement.

2. Contexte 
Aujourd’hui les bailleurs utilisent principalement :
PDF “qui fait quoi” guides locataires livret d’accueil. 

Ces documents :
sont difficiles à comprendre ne permettent pas de diagnostiquer une panne génèrent énormément d’appels. 

La plateforme vise à :
✔ réduire les appels
✔ améliorer la qualification des incidents
✔ rendre les locataires autonomes
✔ réduire les déplacements techniques

3. Base juridique 
Le système repose sur les textes suivants :
Loi du 6 juillet 1989 Décret réparations locatives 1987 Décret charges récupérables 1987 

Principe général :
Situation Responsable entretien courant locataire grosse réparation bailleur contrat d’entretien prestataire 

4. Vision produit 
Le locataire décrit son problème :
texte photo vidéo vocal 

Le système :
1️⃣ analyse
2️⃣ identifie la panne
3️⃣ explique la responsabilité
4️⃣ propose une solution

5. Parcours utilisateur locataire Étape 1 : décrire le problème 
Le locataire peut utiliser :
méthode description texte ex : robinet fuit recherche mot clé vocal dicter photo image vidéo vidéo 
L’IA analyse les données.

6. Schéma interactif du logement 
Le site doit comporter un schéma central interactif.
Fonctions :
zoom tactile déplacement clic sur pièce clic sur équipement 

Exemples :
cuisine séjour salle de bain extérieur 7. Résultat affiché 

Chaque panne affiche une fiche.

🔵 Contrat d’entretien
⚪ À vérifier

Qui paie ? locataire bailleur bailleur récupérable via charges Pourquoi ? 
Référence juridique simple.
Comment diagnostiquer ? 
Check-list simple.
